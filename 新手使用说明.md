# 新手使用说明

你不需要记命令，直接双击这些文件：

## 1. 想看今天有什么机会

双击：

```text
双击运行_扫描全部.bat
```

它会扫描 Binance 现货和U本位合约。

结果在：

```text
reports\latest.md
reports\latest.csv
```

先看 `latest.md`。

## 2. 只想看合约

双击：

```text
双击运行_只扫合约.bat
```

## 3. 只想看现货

双击：

```text
双击运行_只扫现货.bat
```

## 4. 想计算一笔交易能亏多少、赚多少

双击：

```text
双击运行_计算交易方案示例.bat
```

这个文件默认是 UNIUSDT 示例。

如果要改成自己的币：

1. 右键这个 `.bat` 文件
2. 选择“编辑”
3. 改下面这些参数：

```text
--symbol UNIUSDT      币种
--side long           方向，long是做多，short是做空
--entry 3.25          入场价
--stop 3.12           止损价
--target 3.38         目标价
--equity 1000         本金
--risk-pct 1          单笔最多亏本金的百分比
--leverage 2          杠杆倍数
```

## 5. 模拟下单

双击：

```text
双击运行_模拟下单示例.bat
```

这个不会真实交易，只会显示 `dry_run: true`。

## 6. 真下单在哪里

真下单模板是：

```text
run_order_live_TEMPLATE.bat
```

它默认只是模板。你必须右键编辑，填入 Binance API Key，并确认参数后才可能真下单。

真下单前先看：

```text
LIVE_ORDER_README.md
```

不要用带提现权限的 API Key。
第一次真下单只用很小金额测试。

## 7. 怎么看扫描报告

打开：

```text
reports\latest.md
```

重点看两块：

```text
Long Candidates
Short Candidates
```

含义：

- Long Candidates：偏多观察名单
- Short Candidates：偏空观察名单
- score：分数越高，越值得观察
- RSI太高：可能过热，不要追
- RSI太低：可能超跌，不要追空
- note：提醒你等回踩或反弹失败，不要直接冲

## 8. 最重要的提醒

这个工具只是交易助手，不是自动赚钱机器。

新手建议：

- 先只扫描
- 再做交易计划
- 再模拟交易
- 不要一开始开真实自动下单
- 不要高杠杆
